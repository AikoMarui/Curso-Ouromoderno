# Scratch
https://scratch.mit.edu/

O Scratch é uma ferramenta de programação simples, criada com o intuito de que jovens aprendessem a criar jogos por conta propria.

O scrtach usa de uma linguagem propra com intenção de apenas ensinar como funciona a lógia de programação, como if, variaveis, eventos e etc, usando de blocos para montar seus códigos, dentro do curso, nós tivemos aulas para de desenvolver jogos simples, por exemplo um pega pega com o mouse, onde o professor nos ensinou como funciona a lógica na programação de jogos.

onde ele nos ensinou sobre as variaveis: que são um espaço na memória onde armazenamos valores
sobre os eventos: usados para quando algo aconteça, outra ação vai ser realizada, Exemplo: se jogador toca em enimigo, jogador perde.
também sobre posições X e Y, no caso o X representando a posição horizontal do personagem e Y a posição vertical do peronagem.
também nos ensinou sobre as sprites de um personagem, como criar um personagem, alteralas durante o jogo e etc...

em gera, o scratch foi a ferramenta que mais vimos durante o periodo do curso, é uma ferramenta que ajuda muito a quem quer iniciar na criação de jogos, principalmente crianças e pré-adolecentes.
